# React + Antd

## Description

Login modal template using React and Antd

## 1. Clone the repository

```bash
git clone this-repo
```

## 2. Install dependencies

```bash
yarn
```

## 3. Run the project

```bash
yarn dev
```

## 4. Build the project

```bash
yarn build
```

## 5. Run tests

```bash
yarn test
```

## 6. Run lint

```bash
yarn lint
```
